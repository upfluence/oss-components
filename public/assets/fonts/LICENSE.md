# Vendored fonts

These WOFF2 files are vendored from Fontsource packages and are served locally by
`@upfluence/oss-components` to avoid runtime font CDN requests.

- Inter: `@fontsource/inter@5.2.8`, Copyright 2016 The Inter Project Authors
- Roboto Mono: `@fontsource/roboto-mono@5.2.9`, Copyright 2015 The Roboto Mono Project Authors
- Reddit Sans: `@fontsource/reddit-sans@5.2.7`, Copyright (c) 2020-2023 Reddit, Inc.
- Open Sans: `@fontsource/open-sans@5.2.7`, Copyright 2020 The Open Sans Project Authors
