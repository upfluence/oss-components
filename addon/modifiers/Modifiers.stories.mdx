# Modifiers

According to Ember JS:

<figure>
  <blockquote cite="https://www.huxley.net/bnw/four.html">
    <p>Modifiers are a basic primitive for interacting with the DOM in Ember.</p>
  </blockquote>
  <figcaption>— Ember JS</figcaption>
</figure>


| Modifier               | Description                                                                                | Usage                                                           | Default                        | Optional Params                                                                                             |
| ---------------------- | ------------------------------------------------------------------------------------------ | --------------------------------------------------------------- | ------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| enable-popover         | Adds a Bootstrap's Popover to the element                                                  | `<button {{enable-popover [trigger]}}>Popover</button>`         | Default Trigger : **hover**    |                                                                                                             |
| enable-input-autofocus | Adds an autofocus on the selected input or the first non-disabled input child.             | `<input type="text" {{enable-input-autofocus}}/>`               |                                |                                                                                                             |
| enable-dropdown        | Toggles Bootstrap's dropdown on an element                                                 | `<a {{enable-dropdown}}>Dropdown Trigger link/button</a>`       |                                |                                                                                                             |
| on-click-outside       | Triggers the passed callback function when a user clicks outside the element               | `<div {{on-click-outside callbackFunction}}>Floating Div</div>` |                                |                                                                                                             |
| on-bottom-reached      | Triggers the passed callback function when a user reaches the bottom of the target element | `<div {{on-bottom-reached callbackFunction}}>Content</div>`     | Default `offset` of 20px       | `offset:number` <br/><br/> eg: `{{on-bottom-reached callbackFunction offset=100}}`                          |
| on-top-reached         | Triggers the passed callback function when a user reaches the top of the target element.   | `<div {{on-top-reached callbackFunction}}>Content</div>`        | Default `offset` of 20px       | `offset:number` <br/><br/> eg: `{{on-top-reached callbackFunction offset=100}}`                             |
| scroll-shadow          | Add bottom shadow for scrollable element                                                   | `<div {{scroll-shadow color="field"}}>`                         | color: "default" (white)       |                                                                                                             |
| text-carousel          | Displays each provided label over time and keeps the last one visible by default           | `<div {{text-carousel this.labels}}></div>`                     | interval: 3000<br/>loop: false | `interval:number`<br/>`loop:boolean` <br/><br/> eg: `{{text-carousel this.labels interval=3000 loop=true}}` |
