# Variables

Variables are used in .less files to standardize css values. They are represented by css variables.

This is an example how to use
Upfluence css variables:

`.your-css-class {
css-attribute: var(--custom-vaiables);
}`
`<div class="your-css-class"></div>`

## Box shadow

The following values represent box shadow level. The following values represent the 4 different box-shadow levels: xs, sm, md, lg.

| Name            | Class          |
| --------------- | -------------- |
| --box-shadow-xs | .box-shadow-xs |
| --box-shadow-sm | .box-shadow-sm |
| --box-shadow-md | .box-shadow-md |
| --box-shadow-lg | .box-shadow-lg |

## Border color

The default border color used in all components. The content separator color is used to separate different elements, for example
to separate lines of a table.

| Name                      | Value                 |
| ------------------------- | --------------------- |
| --color-border-default    | var(--color-gray-200) |
| --color-content-separator | var(--color-gray-100) |

## Border radius

The following values represent border radius level. There are 2 types of levels: sm, md.

| Name               | Value | Class                  |
| ------------------ | ----- | ---------------------- |
| --border-radius-sm | 4px   | .border-radius-default |
| --border-radius-md | 8px   | .border-radius-md      |

## Spacers

These are variables that can be used alongside size css attributes (padding, margin, etc.).

They range from 0 to 60 incremented each time by 6 (for more precision, 3 and 9 are also available).

| Variable        | Value                                               |
| --------------- | --------------------------------------------------- |
| --spacing-px-XX | Where XX is a number that represents the pixel size |

These spacers are also used to define margin & padding classes to use directly in the templates :

| Class                 | Value                                               |
| --------------------- | --------------------------------------------------- |
| .margin-px-XX         | Where XX is a number that represents the pixel size |
| .margin-right-px-XX   |                                                     |
| .margin-left-px-XX    |                                                     |
| .margin-top-px-XX     |                                                     |
| .margin-bottom-px-XX  |                                                     |
| .padding-px-XX        |                                                     |
| .padding-left-px-XX   |                                                     |
| .padding-right-px-XX  |                                                     |
| .padding-top-px-XX    |                                                     |
| .padding-bottom-px-XX |                                                     |
