# Typography

  type: {

  },
  weight: {
    regular: '400',
    semibold: '600',
    bold: '700'
  }

export const SampleText = 'The quick brown fox jumps over a lazy dog.';

## Product Font

**Open Sans** is the primary typeface accross Upfluence products and can be used
in 3 different weights:

- Regular: 400
- Semibold: 600
- Bold: 700
