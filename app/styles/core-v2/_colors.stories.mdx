# Colors

New color palette use CSS variables.

Note: The most important variables can be used as html classes, calling them as `background-color-[color]-[weight]`.

E.G. `<div class="background-color-primary-50 font-color-gray-900">`

## Brand

Use for primary actions or elements that communicate the Upfluence brand.

<ColorPalette>
  <ColorItem
    title="Primary"
    subtitle="Primary color"
    colors={{ 'Primary 500': getComputedStyle(document.documentElement).getPropertyValue('--color-primary-500') }}
  />
  <ColorItem
    title="Primary Scale"
    subtitle="Primary color"
    colors={[
      'color-primary-50',
      'color-primary-400',
      'color-primary-500',
      'color-primary-600',
      'color-primary-900'
    ].reduce((acc, v) => {
      const humanized = v
        .split('-')
        .slice(1, v.split('-').length)
        .map((x) => {
          return x.toUpperCase();
        })
        .join(' ');
      acc[humanized] = getComputedStyle(document.documentElement).getPropertyValue('--' + v);
      return acc;
    }, {})}
  />
</ColorPalette>

## Neutral Colors

Neutral colors apply to most backgrounds, borders, and shapes in our experiences. They don’t typically have a meaning associated with them, though they can imply things like disabled states or a delimited group.
